Component Inventory

Every component on this screen.

ShowHero Organism
Photo statesuser photo → community top-voted photo → empty state (artist name large on bg-card, no image)
Height190px fixed. Gradient overlay bottom 60% — ensures artist name legibility regardless of photo brightness.
Photo badge"My Photo" (gold tint) or "Community Photo" (muted) — top right. Attribution matters. Always credit the source.
Artist namePlayfair Display Italic 700 · 24px · anchored bottom-left. Venue · City · Date in DM Mono 9px below.
Upload CTAOn empty state or community photo: subtle "Add your photo" tap zone in top-right. Doesn't compete with content when a photo exists.
RatingBlock Molecule
ChildrenStarRating (reused from Log a Show) + user note in Crimson Pro Italic
Note fontCrimson Pro Italic · 13px — the note is a personal voice. Serif italic signals diary entry, not a review field.
Tap behaviorTapping the block → inline edit mode. No separate edit screen needed for just rating/note.
TagPill Atomic
Variantsstat (gold tint — "3rd time seeing them") · memory (muted — "From Memory")
Stat copyAuto-generated from archive data. Appears when user has 2+ shows for the same artist. First show gets no tag — no "1st time" fanfare, that feels cheap.
Memory tagOnly shows if fromMemory is true. Muted deliberately — never a warning, never prominent. A signal, not a disclaimer.
FullSetlist Organism
ChildrenSongRow (reused from Log a Show) + EncoreDivider
Song fontCrimson Pro Italic · 13px — slightly smaller than Log a Show (14px) to fit the detail context density.
Rare badgeSame component as Log a Show. Consistent visual language across screens.
Count"17 songs" in section header right — gives context before scrolling. Setlist nerds want to know immediately.
EncoreDivider Atomic
PurposeSeparates main set from encore within the setlist. A small detail that die-hard fans will immediately clock as correct.
StyleLine · "ENCORE" label · line. Same muted treatment as YearDivider — structural, invisible when not needed.
