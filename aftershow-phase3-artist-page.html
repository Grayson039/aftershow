Artist Page. Component Spec
ArtistHeroMolecule
StatsLogs (total) · Fans (unique loggers) · Avg Rating · Shows (distinct events logged). 4-up grid.
GenresPulled from Setlist.fm metadata. Max 3 shown in pill row.
Follow btn+ Follow Artist → Following (muted state). Tapping Following shows unfollow confirmation.
CommunityPhotosP1 Feature
PurposeTop 3 fan-submitted photos, community-upvoted. These are the fallback thumbnails for feed cards when a user has no personal photo.
Voting▲ upvote. Photos ranked by total upvotes. Anyone can submit. #1 photo used as card fallback.
Why P1Requires moderation layer, storage, CDN. Designed now because it's load-bearing for the P0 card design (fallback hierarchy). Can ship with gradient placeholders at launch.
PlaceholdersConcert-colored gradients in MVP. Swapped to real photos when P1 ships.
ShowHistoryCardAtomic
DateMonth + Day in Syne 800. Gold day number. Month in DM Mono muted.
Tap→ Show Page for that specific event (aggregated setlist from all fan logs)
Log countHow many Aftershow users logged this specific show. Social proof signal.
FanStatsMolecule
Breakdown1 show / 2 shows / 3–5 / 6+ — shows the superfan distribution. Gold bars, descending opacity.
Lead copyPlayfair Display Italic: "3,841 people have logged a SOAD show." — editorial, not a dashboard widget.
MostLoggedSongsMolecule
SourceSetlist.fm API + Aftershow fan logs. Ranked by % of shows where song appeared in fan setlists.
StyleCrimson Pro Italic for song titles. Frequency bar + percentage. Same RARE badge applies if song appears infrequently.
InsightChop Suey! at 97% — SOAD fans will immediately recognize this is accurate. Credibility signal.
